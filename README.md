#  Анализ эффективности маркетинга интернет-магазина

##  О проекте
Анализ маркетинговой эффективности интернет-магазина на основе данных о продажах и рекламных кампаниях.

##  Ключевые метрики
- Общая выручка: €850,250
- Количество заказов: 7,420
- Средний чек: €114.60
- ROI email-рассылок: 320%

##  Основные выводы
1. Email-маркетинг - самый эффективный канал (ROI 320%)
2. VIP клиенты (7%) приносят 40% выручки
3. Мобильные пользователи конвертируются на 35% лучше
