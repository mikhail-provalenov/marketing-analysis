#  Анализ продаж Superstore

##  О проекте
Анализ данных о продажах интернет-магазина Superstore за 2014-2017 годы. Проект включает анализ ключевых метрик, выявление трендов и сегментацию клиентов.

##  Ключевые метрики

| Метрика             | Значение         |
|---------------------|------------------|
| Общая выручка       | $2,297,200.86    |
| Общая прибыль       | $286,397.02      |
| Всего заказов       | 5,009            |
| Уникальных клиентов | 793              |
| Уникальных товаров  | 1,862            |
| Средний чек         | $229.86          |

##  Основные выводы

###  Лидеры по категориям
- Самая продаваемая категория: Technology
- Самый прибыльный регион: West
- Средняя маржа: 12.5%

###  Анализ клиентов
- Крупные клиенты (720 чел.) приносят 99% выручки
- Средние клиенты (59 чел.) - 0.8% выручки  
- Малые клиенты (14 чел.) - 0.03% выручки
- Только 0.6% клиентов совершают разовые покупки

###  Топ-3 самых прибыльных товаров
1. Canon imageCLASS 2200 Advanced Copier - $25,199 прибыли
2. Fellowes PB500 Electric Punch - $7,753 прибыли  
3. Hewlett Packard LaserJet 3310 Copier - $6,983 прибыли

*Топ-3 товара приносят 13.9% общей прибыли*
